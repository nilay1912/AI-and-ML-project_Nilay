# Enhanced Recipe Recommender System 🍳
## Name- Nilay gurdasani (25BAI10563)

A data-science-driven recipe recommendation engine that uses Machine Learning to predict cooking times and provide personalized meal suggestions based on global cuisine data.

## 🚀 Features
- **Smart Filtering:** Filter by cuisine, dietary preference (Veg/Non-Veg), max time, and calorie limits.
- **Scoring Algorithm:** Recipes are ranked using a multi-factor weighted score (Rating, Efficiency, and Health).
- **ML Predictions:** Predicts cooking time for new recipes using Scikit-Learn's `LinearRegression`.
- **Advanced Visualization:** - 3D Regression planes for calorie analysis.
  - Correlation heatmaps of nutritional data.
  - Distribution plots for cuisine-specific ratings.
- **YouTube Integration:** Generates direct cooking tutorial links for recommended dishes.

## 🛠️ Tech Stack
- **Language:** Python 3.x
- **Data Handling:** Pandas, NumPy
- **Machine Learning:** Scikit-Learn
- **Visualization:** Matplotlib, Seaborn
- **Utility:** Urllib (Link Generation), Math
